# Patient Management System (ASP.NET Web Forms)

## 📱 Overview
This is a web-based Patient Management System developed using ASP.NET Web Forms and C#. It allows users to manage patient records with full CRUD functionality.

## ✨ Features
- Add new patient records
- View patient list
- Update patient details
- Delete patient records

## 🛠 Tech Stack
- ASP.NET Web Forms
- C# (.NET Framework)
- SQL Server
- HTML/CSS

## 📌 Project Type
Academic Course Project (.NET Web Development)
